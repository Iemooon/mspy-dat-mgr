微软拼音词库管理器（Windows 绿色版）

使用方法
1. 解压整个 ZIP 文件到本地任意普通目录。
2. 双击 mspy-dat-mgr.exe 启动。
3. 启动后会显示“使用说明”；也可在主界面右下角随时打开。
4. 在词条表格内选中一条或多条词条后，按 Delete 可快速删除；按 Ctrl+Z 可撤销。

安全提示
- 导入前请在微软拼音设置中先备份原有词库。
- 程序不会修改导入的原 DAT 文件。
- 程序不会自动写入 Windows 输入法目录，也不会自动触发微软拼音导入。
- 请通过“导出词库”生成新的 DAT 文件，再由你自行导入并核验。
- workspaces 和 output 目录可能保存你的个人词条；如需清理，请自行确认后删除。

兼容性
本工具已在开发者 Windows 环境中，对两类标准生成 DAT 完成微软拼音实机导入验证。不同 Windows 与微软拼音版本仍可能存在差异，请先用备份和少量测试词条验证。
